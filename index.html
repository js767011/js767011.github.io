CUDA-Accelerated Image Processor

Engineered a high-performance C++ image processing application for massive datasets. Bypassed standard I/O by utilizing pointer casting to map 124MB+ raw binary PPM files directly into dynamically allocated 1D heap memory, explicitly architected for an upcoming parallelized CUDA port.

Tiled Matrix Multiplication (CUDA)

Engineered a heavily optimized CUDA kernel to multiply massive datasets (8192x8192 matrices). Implemented 2D thread blocks and utilized L1 Shared Memory to drastically reduce global memory latency via collaborative tile loading and `__syncthreads()` barrier synchronization.

Parallel 1D Stencil Filter (CUDA)

Developed a highly parallelized 1D stencil algorithm for applying localized mathematical filters across large datasets. Handled complex edge cases using "Halo" ghost cells, pointer arithmetic shifting, and optimized shared memory allocation to prevent segmentation faults during parallel execution.
